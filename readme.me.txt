This program environment is around all America countries. And it gives a data base with a proportional popullation
of each country of America. so this program has the capacity of give a predeterminated data base of people around all
America. Additional the program give the posibility to create new people, search a person, delete a person or
update a person.

This program solve the problem to manage eficiently a a data base, and do the CRUD methods on it. 
with the additional option of generate an amount of people in the data base, according with the context.

The solution of this problem is defined in docs folder.

Tech Spec
lenguaje: java 8
SO: Windows
develop environment: Eclipse and Itellij IDEA
